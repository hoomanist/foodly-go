# foodly-go
foodly rewritten in go

foodly is a social network for foods

## running

using docker:

``` 
# docker pull hoomanist/foodly

# docker run -d -p 5000:5000 hoomanist/foodly
```

by hand :

``` 
$ git clone https://github.com/hoomanist/foodly-go

$ go build 

$ ./foodly
```
